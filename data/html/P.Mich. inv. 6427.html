Notes

  • Fragment 1 recto, line 0: Cursive script written above the line adding to the quote of Is 40:16 in lines 8-9.
  • Fragment 1 recto, line 1: Lines 1-4 have some relationship with Dan 2.22 (Theodotion) and Sir 23:19.
  • Fragment 1 recto, line 3: Possibly related to Is 59:9 (from darkness into daybreak/brightness).
  • Fragment 1 recto, line 4: Note that αχραντε is often used liturgically in conjunction with παρθενε to refer to Mary as the unspotted/unstained/undefiled virgin.
  • Fragment 1 recto, line 6: See Is 66:1
  • Fragment 1 recto, line 7: See Hermas, Mandate 1.1
  • Fragment 1 recto, line 8: See Hermas, Mandate 1.1. See Is 40:16; see also Song 1:3; 4:10
  • Fragment 1 recto, line 9: See Is 40:16; see also Song 1:3; 4:10
  • Fragment 1 recto, line 9a: According to Gronewald, the cursive text above line 1 is intended to be read here.
  • Fragment 2 recto, line 1: See 2Ti 4:17
  • Fragment 2 recto, line 3: See Mt 6:11; James 2:15
  • Fragment 2 recto, line 4: See Php 4:3; Rev 3:5; 13:8; 20:15
  • Fragment 2 recto, line 5: See Hermas, Mandate 9.5
