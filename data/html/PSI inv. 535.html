Notes

  • Fragment 1 recto, line 2: For κρεισεως read κρισεως
  • Fragment 1 recto, line 3: For γεγονασει read γεγονασι. For φονευται read φονευτης, see 2Ki 9:31; Prov 22:13.
  • Fragment 1 recto, line 4: Reconstruction suggested by Naldini (Aegyptus 38). For οτει read οτι.
  • Fragment 1 recto, line 5: For ερε ερε read αιρε αιρε, see Luke 23:18; John 19:15. Reconstruction suggested by Naldini (Aegyptus 38).
  • Fragment 1 recto, line 6: For επει read επι. Naldini notes ενδοξα could possibly be εν δοξη, but prefers to read a form of the verb ενδοξαζω here.
  • Fragment 1 recto, line 8: For βασειλευς read Βασιλευς
  • Fragment 1 recto, line 9: For [.]νωρειδομενος Naldini (Aegyptus 38) suggests possibly [α]νω (ε)ρειδομενος, which is what is translated.
  • Fragment 1 recto, line 10: For περειπολ[ο]υντα read περιπολ[ο]υντα
